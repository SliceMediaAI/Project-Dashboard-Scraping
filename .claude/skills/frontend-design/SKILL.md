# Frontend Design Skill

## Trigger
Invoke this skill when the user asks to build web components, pages, or applications.

## Core Purpose
Generate creative, polished code that avoids generic AI aesthetics. Create production-grade frontend interfaces with distinctive, high-quality design.

## Design Process
Before writing code, complete these steps:
1. **Contextual Analysis**: Understand purpose, audience, and technical constraints
2. **Aesthetic Commitment**: Select a bold directional tone (brutalist, maximalist, retro-futuristic, etc.)
3. **Differentiation Strategy**: Identify what makes the design memorable

## Implementation Standards
Code must be:
- Functional and production-ready
- Visually striking with intentional aesthetic direction
- Meticulously refined across typography, color, motion, composition, and visual details

## Key Aesthetic Priorities
Avoid "generic AI-generated aesthetics" and overused defaults:
- No common/overused font families
- No purple gradient patterns
- No predictable layouts

Instead emphasize:
- Unexpected compositional choices
- Distinctive typography pairings
- Cohesive thematic execution tailored to specific context

Design complexity should match the vision — maximalist designs warrant elaborate code, while minimalist approaches demand precision and restraint.
