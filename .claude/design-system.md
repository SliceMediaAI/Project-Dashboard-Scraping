# Design System - Slice Media (slicemedia.nl)

## Kleuren
- **Achtergrond**: `#0a0a0a` (bijna zwart)
- **Achtergrond secundair**: `#0d0d0d`
- **Tekst primair**: `#ffffff`
- **Tekst secundair**: `rgba(255,255,255,0.56)`
- **Accent**: `#d55012`
- **Borders**: `rgba(255,255,255,0.10)`

## Typografie
- **Primair font**: Inter (400, 700, 900)
- **Display font**: Instrument Serif (italic, 400)
- **Fallback**: Satoshi (400, 500, 700, 900)
- **Line height**: 1.2em

## Layout
- **Max breedte**: 1200px
- **Padding hero**: 160px verticaal, 64px horizontaal
- **Grid**: 2 kolommen, 16px gap
- **Spacing schaal**: 8 / 16 / 24 / 40 / 64px

## UI Elementen
- **Cards**: donkere achtergrond, 1px border (wit 10% opacity), 24px border-radius
- **Knoppen**: 12px padding, flex gecentreerd
- **Navigatie**: fixed, links uitgelijnd
- **Ronde elementen**: 500px border-radius

## Visuele Stijl
- Dark mode als primaire stijl
- Blur effecten op achtergrond gradients (60-70px)
- Conic en radiale gradients als accenten
- Opacity lagen voor diepte (6%, 8%, 77%)
- Modern, minimalistisch, tech-forward
- Warm oranje accent voor highlights
